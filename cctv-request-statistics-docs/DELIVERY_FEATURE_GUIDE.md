# คู่มือฟีเจอร์ "การส่งมอบข้อมูล" สำหรับเจ้าหน้าที่

## ฟีเจอร์นี้คืออะไร?

เป็นฟังก์ชันใหม่ที่ให้เจ้าหน้าที่บันทึกว่าได้ส่งมอบข้อมูลใดให้ผู้ร้องไปแล้วบ้าง เช่น ไฟล์วิดีโอ ภาพถ่าย หนังสือแจ้งผล โดยบันทึกได้หลายรายการต่อคำร้องเดียว

## ใช้งานตอนไหน?

หลังจากบันทึกคำร้องและตรวจสอบภาพจากกล้องวงจรปิดเสร็จ เมื่อต้องส่งมอบข้อมูลคืนให้ผู้ร้อง ไม่ว่าทางช่องทางใดก็ตาม

## วิธีใช้งาน

1. เข้าไปที่หน้ารายละเอียดคำร้อง (หน้าแก้ไขคำร้อง)
2. เลื่อนลงมาที่ส่วน **"การส่งมอบข้อมูล"** จะอยู่ใต้ส่วนหลักฐานแนบ
3. กรอกข้อมูล:
   - **ประเภทข้อมูลที่ส่งมอบ** — เลือกจากรายการ เช่น ไฟล์วิดีโอจากกล้อง ภาพนิ่ง/ภาพถ่ายหน้าจอ รายงานผลการตรวจสอบ หนังสือแจ้งผล
   - **ช่องทางส่งมอบ** — เลือกวิธีที่ใช้ส่ง เช่น รับด้วยตนเอง อีเมล ไลน์ ไปรษณีย์ USB ระบบออนไลน์
   - **ผู้รับ** — ชื่อผู้รับ (ไม่บังคับ)
   - **วันที่ส่งมอบ** — วันที่ดำเนินการ
   - **หมายเหตุ** — เช่น จำนวนไฟล์ ช่วงเวลาของภาพ (ไม่บังคับ)
4. กดปุ่ม **"บันทึกการส่งมอบ"**

ระบบจะแสดงรายการส่งมอบทั้งหมดในตารางด้านล่าง สามารถลบรายการได้หากกรอกผิด

## จัดการประเภทข้อมูลที่ส่งมอบ

หากต้องการเพิ่ม/แก้ไขประเภทข้อมูลที่ส่งมอบ ได้ที่เมนู **"ข้อมูลส่งมอบ"** ในหน้าหลัก ใช้งานเหมือนการจัดการหมวดหมู่และประเภทหลักฐานเดิม

## ในรายงาน

- หน้ารายงานจะแสดงสรุปจำนวนการส่งมอบ แยกตามประเภทข้อมูลและช่องทาง
- ตารางรายการคำร้องมีคอลัมน์ "ส่งมอบ" แสดงจำนวนรายการที่ส่งแล้ว
- การส่งออก Excel และ PDF รวมข้อมูลส่งมอบด้วย

## สำคัญ

- การส่งมอบเป็น **ไม่บังคับ** ไม่กรอกก็ได้ ไม่กระทบ flow หลัก
- บันทึกส่งมอบได้ **หลายครั้ง** ต่อคำร้องเดียว
- ลบรายการได้หากกรอกผิด ระบบจะถามยืนยันก่อนลบ
